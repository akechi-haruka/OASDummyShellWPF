---------------------------------

Dummy Windows shell application.

Serves the only purpose of hiding minimized floating windows if no shell is running. (Alt+Tab still works for debugging purposes)

---------------------------------

Usage:

OASDummyShellWPF.exe [launch_executable]

An optional executable/batch/script file can be specified to be launched after startup.
